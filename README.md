MEMO (alfa)
===========

Co brakuje:

- systemu punktacji
- tablicy wyników (ranking)
- oprawy graficznej (m.in. tłó główne)
- możliwości zmiany zestawu obrazków
- tajmer startuje tylko po aktywacji batona "restart", zamiast za każdym razem po wybraniu rozmiaru tablicy
